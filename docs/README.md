- `docs/project-structure.md`: directory-by-directory ownership.

- `docs/development-workflow.md`: local development and change workflow.
- `docs/testing-and-cicd.md`: local tests + GitHub Actions behavior.
- `docs/troubleshooting.md`: known issues and fast fixes.
- `docs/refactor-review.md`: historical refactor baseline and findings.
